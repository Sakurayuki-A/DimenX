# 新一代视频播放器 - VideoPlayerScreen

## 概述

全新重制的视频播放器，采用现代化设计和优化的性能，为AnimeHUBX提供流畅的视频播放体验。

## 主要特性

### 🎯 核心功能
- **高性能播放**: 基于MediaKit的GPU加速渲染
- **智能缓冲**: 128MB缓冲区，优化网络播放
- **多格式支持**: 支持HLS、MP4等多种视频格式
- **自动提取**: 集成视频链接提取服务

### 🎨 现代化UI
- **渐变控制栏**: 美观的渐变背景，不遮挡视频内容
- **智能隐藏**: 播放时自动隐藏控制栏，3秒后消失
- **响应式设计**: 适配不同屏幕尺寸和方向
- **流畅动画**: 控制栏显示/隐藏的平滑过渡效果

### 🎮 丰富的控制选项
- **播放控制**: 播放/暂停、快进/快退（10秒）
- **进度控制**: 可拖拽的进度条，精确定位
- **音量控制**: 音量滑块和静音切换
- **播放速度**: 0.5x - 2.0x 多档播放速度
- **全屏模式**: 横屏全屏播放支持
- **集数导航**: 上一集/下一集快速切换

### ⌨️ 键盘快捷键
- `空格键`: 播放/暂停
- `←/→`: 快退/快进 10秒
- `↑/↓`: 音量增减
- `F`: 全屏切换
- `M`: 静音切换
- `ESC`: 退出全屏或返回

### 🔧 技术优化
- **内存管理**: 自动清理资源，防止内存泄漏
- **错误处理**: 完善的错误恢复机制
- **网络优化**: 智能请求头和缓存策略
- **性能监控**: 实时播放状态监听

## 使用方法

### 基本用法
```dart
Navigator.push(
  context,
  MaterialPageRoute(
    builder: (context) => VideoPlayerScreen(
      anime: anime,
      episodeNumber: 1,
      sourceRule: sourceRule, // 可选
    ),
  ),
);
```

### 参数说明
- `anime`: 动漫对象，包含标题、集数等信息
- `episodeNumber`: 要播放的集数
- `sourceRule`: 可选的源规则，用于视频链接提取

## 架构设计

### 状态管理
- 播放器状态（播放/暂停、位置、时长等）
- UI状态（控制栏显示/隐藏、全屏模式等）
- 加载状态（提取中、初始化中、错误状态）

### 组件结构
```
VideoPlayerScreen
├── VideoPlayer (主播放器)
├── ControlsOverlay (控制栏覆盖层)
│   ├── TopControls (顶部控制栏)
│   ├── CenterPlayButton (中央播放按钮)
│   └── BottomControls (底部控制栏)
├── LoadingOverlay (加载覆盖层)
└── ErrorOverlay (错误覆盖层)
```

### 生命周期管理
1. **初始化**: 创建播放器实例和控制器
2. **提取**: 使用VideoExtractor提取视频链接
3. **配置**: 设置播放器参数和监听器
4. **播放**: 加载媒体文件并开始播放
5. **清理**: 释放资源和取消监听器

## 错误处理

### 常见错误类型
1. **网络错误**: 连接超时、DNS解析失败
2. **格式错误**: 不支持的视频格式
3. **提取错误**: 视频链接提取失败
4. **播放错误**: 解码失败、文件损坏

### 错误恢复机制
- 自动重试机制
- 用户手动重试选项
- 详细的错误信息显示
- 优雅的降级处理
